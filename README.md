# 💬 Socket.io Chat App

A real-time chat application built with Python (Flask-SocketIO) and WebSockets.

## Features
- Real-time messaging with Socket.io
- Multiple chat rooms
- Private messaging (click on usernames)
- Typing indicators
- Online user list
- Join/leave notifications
- Responsive design
